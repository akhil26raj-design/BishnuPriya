
# Valentine Page (Lucy Edition)

This repository contains a tiny website that asks: **Will you be my Valentine, Lucy?**

## Files
- `index.html` — main page with the question
- `yes_page.html` — success page after clicking **Yes**
- `script.js` — button logic (playful No messages, growing Yes button)
- `styles.css` — styles for main page
- `yes_style.css` — styles for yes page
- `images/img1.gif` — (place your bear-with-flowers GIF here)
- `images/img3.gif` — (place your bear-kissing-panda GIF here)

## How to run locally
Just open `index.html` in a browser.

## GitHub Pages
Push this repo and enable **Settings → Pages** with the `main` branch and root (`/`).
